# Security Policy

## Supported Versions

Specifically identifying which versions of the project are currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

Please report security vulnerabilities by opening an issue on GitHub with the label `security`. We take security seriously and will respond as quickly as possible.
